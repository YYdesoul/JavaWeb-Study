这是一个Java Web的Maven学习项目，每个项目的学习内容如下：



## servlet-01

介绍了如何使用Servlet



## servlet-02

介绍了Servlet的一些进阶方法



## request

介绍了Servlet中的`HttpServletRequest`对象



## response

介绍了Servlet中的`HttpServletResponse`对象



## javaweb-session-cookie

介绍了Servlet中储存全局数据的session和cookie两种方式



## javaweb-jdbc

介绍了使用jdbc连接数据库



## javaweb-filter

介绍了Servlet中的过滤器



##javaweb-jsp

介绍了Java Web开发中的jsp代码
